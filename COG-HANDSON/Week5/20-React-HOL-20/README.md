# React Hands-On Lab 20

## Objective
Demonstrate core React concept: Integrated unit test runners with Vitest asserting count outputs.

## Technologies Used
- React 18
- Vite
- ES6 JavaScript
- Vitest & JSDOM
- React Testing Library

## Folder Structure
```
src/
├── components/
├── pages/
├── services/
├── assets/
├── styles/
│   └── index.css
├── App.jsx
└── main.jsx
```

## Installation
```bash
npm install
```

## Run Commands
```bash
npm run dev
```

## Testing
```bash
npm run test
```

## Expected Output
An interactive React application proving the active lab specs.
