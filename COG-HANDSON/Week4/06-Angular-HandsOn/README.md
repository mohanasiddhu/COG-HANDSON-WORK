# Angular Hands-On Client

## Objective
A modern customer dashboard for Apex Retail Banking, demonstrating Angular components, service dependencies, routing, validation, and reactive forms.

## Technologies Used
- Angular 18 (Standalone Components)
- RxJS
- HttpClient
- HTML5 & CSS Grid/Flexbox (Custom Stylesheet)

## Project Structure
- `src/app/services`: Direct access methods to Spring Boot backend resources.
- `src/app/components`: Custom subcomponents for account/loan directories.

## How to Run
First install the Angular dependencies:
```bash
npm install
```

Start the development server:
```bash
npm start
```

## Expected Output
Access client interface on `http://localhost:4200`. Check existing users or request financial loans.
