# ReactJS Hands-On Lab 05

## Objective
Demonstrate hands-on concept: Render role-specific dashboards conditionally upon login.

## Technologies Used
- React 18
- Vite
- ES6 JavaScript


## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Boot Vite server:
   ```bash
   npm run dev
   ```

## Expected Output
A fully responsive, elegant interface displaying the lab's core concepts.
