# ReactJS Hands-On Lab 17

## Objective
Demonstrate hands-on concept: Integrate unit testing configurations with Vitest and verify components rendering.

## Technologies Used
- React 18
- Vite
- ES6 JavaScript
- Vitest & JSDOM
- React Testing Library

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Boot Vite server:
   ```bash
   npm run dev
   ```
3. Run unit tests:
   ```bash
   npm run test
   ```

## Expected Output
A fully responsive, elegant interface displaying the lab's core concepts.
