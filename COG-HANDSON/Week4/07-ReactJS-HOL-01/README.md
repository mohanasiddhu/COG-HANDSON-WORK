# ReactJS Hands-On Lab 01

## Objective
Demonstrate hands-on concept: Dynamic greetings and timestamps using state/effects.

## Technologies Used
- React 18
- Vite
- ES6 JavaScript


## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Boot Vite server:
   ```bash
   npm run dev
   ```

## Expected Output
A fully responsive, elegant interface displaying the lab's core concepts.
